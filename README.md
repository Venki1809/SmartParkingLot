# SmartParkingLot System

This is a Java-based Parking Lot Management System with a JavaFX user interface and MySQL database backend.

## Features

- **Login Window:** Allows admin to log in.
- **Admin Dashboard:** Options for entering a vehicle, exiting a vehicle, and admin control.
- **Vehicle Entry Form:** Collects vehicle details and stores them in the ParkingLot table.
- **Vehicle Exit Form:** Updates the vehicle exit time and calculates the parking fee.
- **Admin Control:** Provides options for managing the system and viewing the parking history.

## Technologies Used

- **Java**
- **JavaFX** for the user interface
- **MySQL** for the database
